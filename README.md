# FreshFind

A marketplace connecting customers with local vegetable cart vendors.

## Project Structure

- `backend/` - Node.js/Express API server
- `customer-app/` - React Native app for customers
- `seller-app/` - React Native app for vendors
- `admin-dashboard/` - React web dashboard for admin

## Tech Stack

- React Native (Expo)
- Node.js + Express
- PostgreSQL
- JWT Authentication

## Learning Goals

- React Native & JavaScript
- Backend development
- Database design
- Authentication & sessions
- Git & GitHub workflow
